# Iris Flower Classification

## Project Overview
This project uses machine learning to classify Iris flowers into three distinct species (Setosa, Versicolor, and Virginica) based on their sepal and petal dimensions. 

## Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

## Model Performance
* **Algorithm Used:** Random Forest Classifier
